# Contributing

This repository accepts issues and pull requests for the Persistent Agentic Memory Architecture Specification (PAMSPEC) Internet-Draft and supporting review artifacts.

Contributions should:

- preserve provider, runtime, framework, storage, transport, and implementation independence;
- avoid adding implementation code for runtimes, servers, SDKs, databases, vector stores, or user interfaces;
- describe normative changes explicitly;
- include security and privacy impact notes for architectural changes;
- use decision records for unresolved or consequential design choices;
- keep generated Internet-Draft outputs synchronized with the editable source when tooling is available.

Pull requests that introduce normative requirements should explain the interoperability, conformance, security, and privacy consequences.

## Licensing

Documentation contributions are intended for CC BY 4.0 licensing. Technical artifact contributions are intended for Apache-2.0 licensing. Internet-Draft contributions are also subject to applicable IETF contribution provisions if submitted.
